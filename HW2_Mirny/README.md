# HW2_Mirny - Calculator

## Description

This project is Homework 2 for the "Bioinformatics Institute" bioinformatics program.

The program `calculator.py` reads a mathematical expression in the format:
`<number_1> <operator> <number_2>`


where `<operator>` is one of: `+`, `-`, `*`, `/`.
The program calculates the result and prints it to the screen.

## Examples
5 - 3 -> 2

10 / 2 -> 5.0
